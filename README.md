# TodoMVC noscript-react

[**noscipt-react**](https://github.com/yandex-ui/noscript-react)

![](./preview.png)

## Установка

```
$ make install
```

## Запуск приложения

```
$ make server
```

После запуска сервера, приложение станет доступным по адресу `http://localhost:8080`

## Development

```
$ make watch
```

## Сборка YATE

```
$ make yate
```
